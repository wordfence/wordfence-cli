# Wordfence CLI Subcommands

Wordfence CLI provides a variety of subcommands to support different use cases. The following subcommands are currently available:

- **[malware-scan](malware-scan/)** (scan files for malware)
- **[vuln-scan](vuln-scan/)** (scan one or more WordPress installations for known vulnerabilities
- **[db-scan](db-scan/)** (scan one or more WordPress databases for malicious content)
- **[remediate](remediate/)** (remediate malware by restoring known files)
- **[count-sites](count-sites/)** (count the number of WordPress sites at one or more paths)
